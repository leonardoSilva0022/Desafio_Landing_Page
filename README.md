**Sua Próxima Viagem:** Busan, a estrela em ascensão da Coreia do Sul

Um site simples de landing page para promover destinos turísticos inspiradores. O projeto é feito com HTML e CSS puros, ideal para aprendizado, prototipagem ou como base para um portfólio.

## 🌟 Visão geral
- Página única (single-page) com seções de **Início**, **Destinos** e **Contato**.
- Layout responsivo e imagens de exemplo incluídas na pasta `assets/`.

## ▶️ Como executar
1. Clone ou faça download deste repositório.
2. Abra o arquivo `http://127.0.0.1:5501/index.html#inicio` no seu navegador.


## 📁 Estrutura do projeto
- `index.html` - página principal.
- `style.css` - estilos usados na página.
- `assets/` - imagens e recursos usados no site.

## ✨ Personalização
- Altere o texto em `<section id="destinos">` para adicionar seus próprios destinos.
- Substitua as imagens em `assets/` por fotos de viagens reais.
- Edite o cabeçalho e o rodapé para refletir suas informações de contato.
